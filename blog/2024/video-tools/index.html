我的视频制作工具箱

2024/03/27 | 字数1083 | 阅读3分钟

这是一期工具介绍，做UP主一段时间了，介绍一下用到的视频制作工具。主力机是一台2023年的MacMini（16G+512G），用于日常视频制作，剪辑，代码开发，文档编写等，还有一台备用机15款的MacBookPro（16G+256G，最后一款带发光Logo的Mac），外出时，写写文档和代码，还能用，不过也快退休了。

构思&脚本编写

飞书：字节跳动的在线文档平台，功能强大，简单易用，支持MarkDown，代码格式化，软件流程图，思维导图等。用来构思视频内容，编写解说脚本，存储一些视频中用到的代码等，免费Web平台。

视频制作

演示工具

KeyNote，水果家的产品发布会演示工具，界面简洁，动画效果一流。系统自带，免费软件。

录屏工具

Omi录屏专家，支持屏幕录制，区域录制，APP录制等。不过笔者平时只用到区域录制的功能。AppStore下载，收费软件。

键盘工具

KeyCastr，显示键盘按键，方便观众看到UP主是的键盘输入信息，免费软件，Homebrew下载即可。

brew install --cask keycastr

鼠标工具

Cursor Pro，给鼠标加一个光环，方便观众看到鼠标位置。AppStore下载，收费软件。

视频剪辑

视频剪辑

水果家的FCP（Final Cut Pro），是真的贵呀，也真的专业，你能想到的功能基本都有，完美适配Mac系统。

字幕制作

文字是提前写好的，用【剪映】做一次文字语音对齐，导出SRT文件即可，如果想要把SRT字幕文件输入到FCP中制作带字幕的视频，就需要用到 srt2fcpxml 这个命令行工具做转换了（这是一个笔者基于一个开源软件改过的）。

语音合成

TTSMaker，用于文字转语音，各种音色都有，支持简单的语速调节、停顿时间调节等配置，免费Web APP。

至于为什么不用笔者原声呢？主要是这样效率更高，方便后期修改。另外笔者的声音也是有点难以形容，效果并不比生成的语音好。

归档

百度云：素材、剪辑项目文件、字幕、最终视频等，都放在这里了。方便跨设备资源共享。

垃圾清理

function v.fcp.gc() {
  find . -type d -name "Render Files" -print0 |
    while IFS= read -r -d '' one; do
      echo "$one"
      rm -rf "$one"
    done
  find . -name ".DS_Store" -print0 |
    while IFS= read -r -d '' one; do
      echo "$one"
      rm -rf "$one"
    done
  find . -size 0 -print0 |
    while IFS= read -r -d '' one; do
      echo "$one"
      rm -rf "$one"
    done
}
function v.fcp.gc() { find . -type d -name "Render Files" -print0 | while IFS= read -r -d '' one; do echo "$one" rm -rf "$one" done find . -name ".DS_Store" -print0 | while IFS= read -r -d '' one; do echo "$one" rm -rf "$one" done find . -size 0 -print0 | while IFS= read -r -d '' one; do echo "$one" rm -rf "$one" done }

把上面的代码放到你的.zshrc或者.bashrc中就可以了，然后需要清理的目录下，执行v.fcp.gc，就可以删除垃圾文件啦。
