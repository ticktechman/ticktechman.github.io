白嫖一个github博客系统

2024/03/18 | 字数440 | 阅读1分钟

你现在看到的这个博客系统，就是托管在了github上，对于纯静态网站来说，github可能是性价比非常高的选择，非常适合纯纯的写作者朋友。

为什么选择Github

  • 免费（穷），域名+证书+带宽+服务器+存储，这些通通免费；
  • 简单（当然是相对于Github老鸟来说），一个git库就是一个网站；
  • 不需要实名认证和备案，虽然如此，也不要放违规内容上去；
  • 历史数据留存，不用单独开库保存历史记录；

前提

  • 熟悉Git基础操作
  • 了解Github的使用流程
  • 了解一些基础的HTML、CSS、JS编程

如何开通

  1. 注册一个github账号 https://github.com/signup（老用户直接跳到第二步）；
  2. 新建仓库，命名为{your-user-name}.github.io，这里的{your-user-name}是你的github用户名；
  3. 上传你的网站内容（HTML、CSS、JS、图片等）；
  4. 浏览器访问 https://your-user-name.github.io（首次上传需要等待几分钟）；
